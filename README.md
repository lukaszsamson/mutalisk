# Mutalisk

Mutalisk is an Elixir mutation testing library under active milestone-based implementation.

Stryker-compatible JSON reports target the mutation-testing-elements v2 schema and can be viewed with the `@stryker-mutator/mutation-testing-elements` HTML viewer.
